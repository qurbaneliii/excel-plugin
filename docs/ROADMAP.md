# Roadmap

## Completed MVP+

- Excel task pane add-in
- FastAPI backend
- OpenAI backend-only integration
- Ollama local provider integration
- deterministic fallback provider
- formula assistant
- safe cleaning plan and apply flow
- report generation
- basic summary sheet
- provider status endpoint and task pane visibility

## Next Recommended Steps

- Verify the add-in live inside Excel desktop with representative workbooks
- Add integration tests around provider-mode request overrides
- Improve formula generation with more column-aware examples
- Add richer summary sheet aggregations
- Add optional chart creation behind a safe capability check
- Add structured logging without exposing prompts or secrets

## Deferred

- Custom functions implementation
- Enterprise auth
- Deployment packaging
- Advanced dashboarding
